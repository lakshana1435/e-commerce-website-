# e-commerce-website-
It is an mini design of an ecommerce website with basic web development concepts.  
This project is a modern, responsive e-commerce frontend website developed using HTML, CSS, and JavaScript.
It displays fashion products with images, allows users to add products to a cart, and dynamically updates the cart count using DOM manipulation.
The project focuses on UI design, interactivity, and frontend logic, without backend integration.
HTML5 is used to create the structure and layout of the web pages.
CSS3 is used for styling, colors, fonts, and overall visual design.
Flexbox is used to align the header, navigation, and sections properly.
CSS Grid is used to display products in a responsive grid layout.
JavaScript (ES6) is used to add interactivity to the website.
DOM Manipulation is used to update the cart count dynamically.
Event Handling is used to trigger actions like “Add to Cart”.
Responsive Design is implemented to support different screen sizes.
Browser Environment is used to run and test the project without a backend.
